Exits - Le jeu du labyrinthe abandonné

Web design

Figma icon by Icons8

Une aventure terrifiante !

Exits est un site vitrine pour un jeu vidéo. Il y a une page informations, une page d'accueil pour se diriger vers le jeu ainsi qu'une page équipe. Le principe était de créer des maquettes du site sur Figma et d'y intégrer le jeu.
Nous avons choisi de nous baser sur un jeu d'horreur. Plutôt que de réaliser un jeu de rôle nous avons préféré nous baser sur un labyrithe abandonné avec et l'utilisateur décidie d'où il veut aller. Mais il peut aussi perdre s'il rencontre un des monstres de l'hôpital.
La maquette est donc à l'image du jeu, effrayante !
